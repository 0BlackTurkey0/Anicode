# Anicode
Anicode by NTPU2021 - Software Engineering Group 1

## 介紹
我們開發了一款遊戲以程式設計為基礎，並加入競技與娛樂的性質，用來培養程式邏輯與素養。

## 環境

### 開發
`Windows 10`, `Unity 2020.3.22`

### 執行
`Windows 10`, `單一區域私人網路`(雙人模式需求)
